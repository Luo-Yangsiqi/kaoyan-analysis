## 第11页 · PEEHI 综合热度指数构建（熵权法）⭐⭐⭐

### 为什么需要 PEEHI？

> 单一指标（如报名人数）无法全面反映考研热度。
> PEEHI 将参与率、报录比、统考竞争度三个维度合成为一个 [0,100] 的综合指数。
> 熵权法 = 客观赋权法，利用数据自身的信息量大小确定权重，避免主观偏差。

### 核心代码段：熵权法完整实现

```python
# data_analyze/hot_index.py —— 熵权法核心算法
import numpy as np

# ---- Step 1: 定义三个子指标 ----
res["X1_参与率"] = res["报名人数(万)"] / res["普通本科毕业生数(万)"]
res["X2_报录比"] = res["报名人数(万)"] / res["硕士实际录取人数(万)"]
res["X3_统考竞争度"] = res["报名人数(万)"] / (
    res["硕士总招生(万)"] - res["推免保研人数(万,估算)"])

X = res[["X1_参与率", "X2_报录比", "X3_统考竞争度"]].values
n = X.shape[0]

# ---- Step 2: Min-Max 正向归一化 ----
X_min = X.min(axis=0); X_max = X.max(axis=0)
Z = (X - X_min) / (X_max - X_min)

# ---- Step 3: 构造概率矩阵 ----
P = Z / (Z.sum(axis=0) + 1e-12)

# ---- Step 4: 计算信息熵 e_j ----
e = -1.0 / np.log(n) * np.sum(P * np.log(P + 1e-12), axis=0)

# ---- Step 5: 差异系数 → 权重 ----
d = 1.0 - e
weights = d / d.sum()
# 结果：X1_参与率=0.3777  X2_报录比=0.3188  X3_统考竞争度=0.3035

# ---- Step 6: 合成 PEEHI，二次归一化到 [0,100] ----
res["PEEHI_原始"] = np.dot(Z, weights)
res["PEEHI_归一化"] = ((res["PEEHI_原始"] - res["PEEHI_原始"].min())
                   / (res["PEEHI_原始"].max() - res["PEEHI_原始"].min()) * 100.0)
```

### PEEHI 权重与关键数值

| 子指标 | 权重 | 含义 |
|--------|------|------|
| X1 参与率 | **0.3777** (第一) | 本科毕业生中有多大比例选择考研 |
| X2 报录比 | 0.3188 (第二) | 平均每个录取名额有多少人竞争 |
| X3 统考竞争度 | 0.3035 (第三) | 剔除推免后统考通道的真实竞争 |

| 关键节点 | PEEHI | 说明 |
|----------|-------|------|
| 2015 年 | **0**（冰点） | 报名人数低谷+录取率相对高 |
| 2020 年 | 跳升 | 疫情+扩招政策双重冲击 |
| 2022 年 | **100**（峰值） | 报名 457 万+推免挤压+报录比恶化 |
| 2024 年 | **62.6** | 报名首降，回落 37.4 点 |

**关键设计说明**：X4_升学率 已从 PEEHI 中移除——升学率衡量的是供给端响应（"退烧药"），不适合作为热度指数（"体温计"）的构成指标。

```python
# 通用标准化函数
def norm(v):    return (v - np.nanmin(v)) / (np.nanmax(v) - np.nanmin(v)) * 100
def norm_rev(v): return (np.nanmax(v) - v) / (np.nanmax(v) - np.nanmin(v)) * 100

# S1 升学基数压力 —— 本科生越多+读研意愿越强 → 考研基数越大
S1 = (norm(df["普通本科毕业生数(万)"]) + norm(df["读研比例(%)"])) / 2

# S2 就业市场压力 —— 薪资增长停滞+就业率下降+失业率上升 → 就业驱动考研
g = np.full(len(rs), np.nan)
g[1:] = (rs[1:] / rs[:-1] - 1) * 100  # 实际薪资增长率
S2 = np.nanmean([norm(unemp), norm_rev(emp_rate), norm_rev(g)], axis=0)

# S3 招生政策供给 —— 硕士扩招+专硕占比提升+政策刺激 → 供给端驱动
S3 = (norm(df["硕士实际录取人数(万)"]) + norm(df["专硕占比(%)"])
      + df["政策哑变量(2020后=1)"] * 100) / 3

# S4 考公替代压力 —— 国考过热→分流考研人群（替代效应）
S4 = (norm(df["报名过审人数(万)"]) + norm(df["报录比(X:1)"])) / 2
```

### 核心代码段

```python
# data_analyze/grey_relational.py —— 灰色关联度完整计算
Y = peehi_df["PEEHI_归一化"].values  # 参考序列
X = np.column_stack([S1, S2, S3, S4])  # 比较序列

# 标准化到 [0,1]
Y_norm = (Y - Y.min()) / (Y.max() - Y.min())
X_norm = (X - X.min(axis=0)) / (X.max(axis=0) - X.min(axis=0))

# 差序列 → 关联系数
diff = np.abs(X_norm - Y_norm.reshape(-1, 1))  # (15, 4)
rho = 0.5  # 分辨系数
coef = (diff.min() + rho * diff.max()) / (diff + rho * diff.max())  # 逐年

# 灰色关联度 = 各年关联系数均值
grades = coef.mean(axis=0)  # 最终排序依据
```

### 核心代码段

```python
# data_trend_simulate/scenario_simulation.py

# 锚定校准：从2024年真实PEEHI自然延伸
score_2024 = np.dot(X_hist[-1], w_vec)  # 2024年机制得分
peehi_2024 = PEEHI_hist[-1]             # 2024年真实PEEHI

def calibrated_peehi(S_mat):
    """以2024为锚点，将机制得分变化映射为PEEHI变化"""
    score = np.dot(S_mat, w_vec)
    return np.clip(peehi_2024 + (score - score_2024), 0, 100)

def make_scenario(multipliers=None, diversion_elasticity=None):
    dS = dS_avg.copy()
    for key, value in (multipliers or {}).items():
        dS[key] *= value
    S_future = simulate(dS.values, last_S)  # 线性外推
    PEEHI_future = calibrated_peehi(S_future)
    if diversion_elasticity is not None:
        # 考公分流对PEEHI的额外折减
        lam = w_vec[3] + diversion_elasticity
        PEEHI_future -= lam * (S_future[:, 3] - S4_2024)
        PEEHI_future = np.clip(PEEHI_future, 0, 100)
    return S_future, PEEHI_future

# 四种情景
S_base, PEEHI_base = make_scenario()  # 基准延续
S_emp, PEEHI_emp = make_scenario({"S2": 1.5})  # 就业压力加剧
S_split, PEEHI_split = make_scenario({"S4": 1.3}, diversion_elasticity=0.10)  # 考公分流
S_policy, PEEHI_policy = make_scenario({
    "S1": 0.2, "S2": -0.4, "S3": 0.0, "S4": -0.2  # 综合治理降温
})
```

### 四种情景结果

| 情景 | 2030年PEEHI | 方向 | 参数设定 |
|------|------------|------|----------|
| 🔵 基准延续 | **69.0** | 高位震荡 ~62→69 | 维持当前变化率 |
| 🔴 就业加剧 | **69.7** | 微升 | S2×1.5（就业市场进一步恶化） |
| 🟣 考公分流 | **62.3** | 微降 | S4×1.3 + 分流弹性 0.10 |
| 🟡 综合治理 | **58.2** | 持续下行 | S1×0.2、S2×-0.4、S3=0、S4×-0.2 |

> ⚠️ **重要声明**：这不是预测，而是探索性推演。核心假设包括：灰色关联权重在未来保持稳定、四类机制按近五年变化率线性外推、S1/S3 已接近 [0,100] 上限存在结构性制约。

##结论与建议

### 核心发现

1. **2024 年是考研热度拐点**：PEEHI 从 2022 年峰值 100 回落至 62.6，报名人数首降 ↓7.6%
2. **驱动机制多元均衡**：灰色关联度排序为 招生政策(0.766) > 升学基数(0.740) ≈ 就业市场(0.734) > 考公替代(0.624)，但差距不大
3. **考研热度接近结构性顶部**：基准情景 2030 年 PEEHI 仅微升至 69，远低于 2022 年的 100
4. **唯有综合治理能实现持续降温**：就业改善 + 招生稳定 + 多元发展路径协同 → 2030 PEEHI=58.2
5. **推免挤压统考空间**：推免比例从 10%→14%，统考名额被学硕缩量 + 推免扩张双向挤压

### 政策建议

```
🎯 短期：稳定专硕招生预期，避免供给端过度刺激
🎯 中期：改善本科毕业生就业质量，降低"被动考研"比例
🎯 长期：拓宽多元发展路径（考公、创业、职业教育），降低路径依赖
```
